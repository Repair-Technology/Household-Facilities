## 问题定义：

1. Dell笔记本电脑开机时连续二声报警音，即未检测到内存
2. 环境描述：Dell Inspiron 15R N5110

## 处理过程：检查内存

1. 拆开笔记本电脑，露出内存即可
2. 拿下内存，使用橡皮擦拭金手指
3. 安装内存，开机测试
4. 如果未能解决，更换内存
