## 场景定义：

1. 电脑启动时显示并不存在的第二个或多个操作系统选项，需要删除它
2. 测试OS：Windows 7

## 处理过程：

1. windwos+R
2. 输入msconfig-确定
3. 系统配置窗口-引导
4. 选择非当前OS-删除-确定
5. 重启
6. 系统启动正常
